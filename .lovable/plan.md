## 1. Product Description (Elevator → Long-form)

### One-liner
**Roomly is the all-in-one cloud Property Management System that helps independent hotels, B&Bs, and small chains automate reservations, prevent overbookings, and run daily operations from a single dashboard.**

### 30-second pitch
Roomly replaces spreadsheets, paper checklists, and disconnected booking tools with one modern PMS. Manage reservations, housekeeping, invoicing, channel sync (Booking.com, Airbnb via iCal), guest emails, night audits, and analytics — from any device. Built on enterprise-grade infrastructure with row-level security, atomic booking locks (zero overbookings), and real-time updates across your team.

### Long-form description
Roomly PMS is a **multi-tenant, cloud-based hotel management platform** designed for properties that have outgrown spreadsheets but don't want the complexity or price tag of legacy systems like Opera or Cloudbeds. It unifies the **entire guest journey** — from booking and check-in to invoicing and post-stay communication — while giving managers real-time visibility into occupancy, revenue, and staff operations.

**Who it's for:**
- Independent hotels (10–80 rooms)
- Boutique B&Bs and guesthouses
- Vacation rental operators with multiple units
- Small hotel chains needing per-property isolation

**Why it wins:**
- **No overbookings, ever** — atomic database locks during reservation creation
- **Channel sync built in** — bidirectional iCal with Booking.com, Airbnb, Vrbo
- **Automated operations** — night audit reports, no-show detection, housekeeping boards
- **Real branded emails** — confirmations, check-ins, invoices sent from your own domain
- **Multi-property ready** — superadmin panel for chains and management groups
